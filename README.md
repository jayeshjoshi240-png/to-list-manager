# 🌌 Cosmic Task Dashboard

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E)

A stunning, database-free task management dashboard featuring a premium glassmorphism UI, real-time progress tracking, and category tagging. Built entirely with vanilla web technologies, this application runs flawlessly in the browser and uses local storage to ensure your tasks are never lost.

## ✨ Features

* **📊 Live Dashboard Metrics:** Automatically tracks and calculates your total, completed, and pending tasks, displaying them alongside a dynamic visual progress bar.
* **🏷️ Mission Tags (Categorization):** Assign tasks to specific focus areas (Core System, UI/UX, Database, General) visualized by glowing, color-coded pills.
* **🌗 Cosmic Glassmorphism UI:** A deep-space aesthetic featuring frosted glass panels, custom scrollbars, and ambient floating background elements. Includes a seamless Light/Dark mode toggle.
* **💾 Zero-Database Persistence:** Saves all tasks, categories, and theme preferences directly to the browser's `localStorage` instantly. 
* **⚡ Ultra-Lightweight:** No frameworks, no dependencies, and no build tools required. 

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modern CSS variables, Backdrop-filter, CSS Animations)
* **Logic:** Vanilla JavaScript (ES6+)
* **Storage:** Browser `localStorage` API
